El DOM o Document Object Model es una de las innovaciones que más ha influido en el desarrollo de las páginas web dinámicas y de las aplicaciones web más complejas.

El DOM permite a los programadores web acceder y manipular las páginas HTML como si fueran documentos XML. De hecho, el DOM se diseñó originalmente para manipular de forma sencilla los documentos XML.

El DOM transforma todos los documentos HTML en un conjunto de elementos llamados nodos, que están interconectados y que representan los contenidos de las páginas web y las relaciones entre ellos. Por su aspecto, la unión de todos los nodos se llama “árbol de nodos”.

El DOM se ha convertido en una utilidad disponible para la mayoría de lenguajes de programación como JavaScript.
